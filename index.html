Machine Learning Concepts

Q. 1 a) Supervised, Unsupervised aur Reinforcement Learning ka Difference aur Example

Supervised Learning: Labeled dataset hota hai. Example: Email Spam Detection

Unsupervised Learning: Labels nahi hote. Example: Customer Segmentation

Reinforcement Learning: Agent environment me actions leta hai. Example: Chess-playing AI

Q. 1 b) Empirical Error ka Significance in Machine Learning

Empirical error training dataset par model ka error hota hai.
Example: 100 images classify ki, 20 galat → Error = 20%

Q. 2 a) Classifiers Compare Karne Ke Liye Statistical Tests

T-test: Do classifiers ke accuracy compare karta hai.

Wilcoxon Signed-Rank Test: Small datasets ke liye.

McNemar’s Test: Binary classification me.

ANOVA: Multiple classifiers ke comparison me.

Q. 2 b) PAC Learning aur VC Dimension ki Importance

PAC Learning: Learning efficiency define karta hai.

VC Dimension: Classifier ki complexity define karta hai.

Q. 3 a) K-Fold Cross Validation ka Importance

Dataset ko K parts me divide kar ke har part ko alag-alag train-test set banaya jata hai.

Q. 4 a) Support Vector Machine (SVM) Algorithm

Best hyperplane choose karta hai jo maximum margin maintain kare.

Q. 4 b) Backpropagation Algorithm for Multilayer Perceptron

Gradient Descent aur Chain Rule ka use karke weights update hote hain.

Q. 5 a) XOR Gate Problem Cannot Be Solved Using Perceptron (True or False?)

✅ True! Kyunki XOR problem linearly separable nahi hai.

Q. 5 b) Curse of Dimensionality in Machine Learning

Jab dimensions badhte hain, to data sparse ho jata hai aur distance unreliable ho jati hai.
